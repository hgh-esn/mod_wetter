Wetter Konfiguration

Modul zur Anzeige der Wetterdaten von Wetter.com für JOOMLA ab v1.7.0

Konfiguration

Um das Modul nach erfolgreicher Installation zu konfigurieren, wechseln Sie in der Administrations-Oberfläche über das Menü; Erweiterungen->Module zur Auflistung aller installierten Module. Selektieren Sie das Modul Wetter. Klicken Sie hierzu einfach auf den Modulnamen.

Nun sehen Sie die Seite Modul: Bearbeiten. Hier konfigurieren Sie das Wettermodul ganz nach Ihren Vorstellungen. Besonders wichtig sind die Einstellungen in der Box Parameter!!

Um Ihre Einstellungen zu sichern klicken Sie auf Speichern (das Diskettensymbol) und vergessen Sie bitte nicht, das Modul abschließend zu aktivieren. Wechseln Sie zum Frontend und kontrollieren Sie, ob das Modul wunschgemäß arbeitet.

PLZ/World-/WMO-ID

PLZ: Geben Sie einfach Ihre Postleitzahl an.
World-ID: Es kann passieren, dass sich mehrere Ortschaften 
eine Postleitzahl teilen. Genauer definieren Sie den
gewünschten Ort dann mit der World-ID.
Um diese herauszufinden begeben Sie sich auf Sie Website 
von Wetter.com und suchen dort die gewünschte Ortschaft.
So liefert eine Suche nach Berlin zunächst die PLZ 14199. 
Lassen Sie sich nun mittels Klick auf den Ortsnamen die 
Wetterdaten anzeigen, so wird hierzu folgende URL aufge-
rufen:
"http://www.wetter.com/v2/?SID=&LANG=DE&LOC=7000&LOCFROM=0202&type=WORLD&id=4788".
Interessant ist der Schluss der URL: "type=WORLD&id=4788".
Die World-ID von Berlin ist also 4788.
(Nehmen Sie lieber die PLZ, falls das bei Ihnen funktioniert ...)
WMO-ID: Hier gilt Ähnliches wie bei der World-ID.
Möchten Sie zum Beispiel auf Ihrer Website die Wetterdaten
von Rhodos anzeigen, finden Sie dazu auf der Website von
Wetter.com die URL:
"http://www.wetter.com/v2/?SID=&LANG=DE&LOC=7000&LOCFROM=7000&type=WMO&name=Rhodos&id=711".
Auch hier interessiert uns der Schlussteil der URL:
"type=WMO&name=Rhodos&id=711".
Auf der Konfigurationsseite des Moduls Wetterdienst wählen
Sie also die Typ-ID "WMO" und tragen als Code die "711" ein.
